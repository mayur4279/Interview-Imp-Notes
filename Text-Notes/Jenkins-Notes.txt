to avoid delay we use CICD  


Continuous Intigartion :- Build --> Testing (code test) --> Merge (merge to main branch)  

Continuous Delivey:- Automatically Release to Repository (means code is ready anytime we can handover to the customer)  

Continuous Deployment: Automatically Deploy To Production (code is deploy into production anytime we can handover to the customer)
